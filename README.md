# Bash scripts
  Small bash scripts that have been created for use in a teaching environment at Huddinge Gymnasium.
  The scripts does use the [git CLI](https://cli.github.com/) to create and work on GitHub repository for the students.   
## Issue script
  A script to create the same issues for several different repository. Does need all the repository's names and that the one that runs the script to be a collaborator if the repository's are private.

For the issues it searches for a folder called `issues` and search for a markdown file with the name given in the issue script.
